# AutoHire-Pro
AI-Powered Resume Matcher and Interview Scheduler
AutoHire Pro is a smart recruitment automation system designed to simplify and accelerate the candidate shortlisting and interview scheduling process. It uses advanced keyword matching between resumes (PDFs) and job descriptions to identify the best-fit candidates. Candidates scoring 70% or higher are automatically shortlisted, stored in a database, and scheduled for interviews within a specified time window.
